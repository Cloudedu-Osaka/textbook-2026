01_intro.ipynb: 円を描く — なぜ Thales か？#

学習目標#

  • 古代ギリシャにおける幾何学の意義を理解する

  • Thales の生涯と彼が解決した問題を知る

  • なぜ「円の直径に対する円周角」が重要なのかを理解する

Thales（タレス）とは#

生涯#

  • 生: 紀元前 624 年（おおよそ）

  • 死: 紀元前 546 年

  • 出身: ミレトス（現在のトルコ）

  • 時代: 古代ギリシャの哲学者・数学者

業績#

Thales は以下の発見で知られています：

  1. 幾何学の父 — 幾何学を「体系的な学問」として確立した最初の人物

  2. Thales の定理 — 「円の直径に対する円周角は直角である」

  3. 電気磁気 — 琥珀（amber）を磨くと異なる物質を引き付けることを発見

  4. 哲学 — 「万物は水からできている」という初期 monism（唯物論）を唱えた

古代ギリシャにおける幾何学#

なぜ幾何学なのか？#

古代ギリシャでは、数学は以下の目的で発展しました：

1. 実用的需要

  • 土地測定（エジプトのナイル川の洪水後の土地区画）

  • 建築（神殿、劇場）

  • 天文学（星座、季節の予測）

2. 哲学的興味

  • 「世界の根本原理は何か？」

  • 「なぜ自然界はこのような形をしているのか？」

  • 「真理とは何か？」

3. 美的関心

  • 円、正多角形の完全性

  • 対称性と調和

Thales の定理の発見#

問題設定#

Thales は以下の問いに直面しました：

「円とは何か？」

円は「中心から等距離にあるすべての点の軌跡」として定義される。

では、円の上にある 3 つの点と円の中心との間に、特別な角度関係があるだろうか？

特別な場合: 直径#

Thales が特に注目したのは、円の直径の端点 (\(A\), \(B\)) と、円周上の任意の点 (\(P\)) によって形成される角 \(\angle APB\) でした。

\[\text{Thales の観察}: \angle APB = 90° \text{ (常に)}\]

なぜこれが重要か？#

  1. 普遍的性質 — すべての円、すべての点に対して成立

  2. 作図への応用 — コンパスと定規のみで直角を作図できる

  3. 建築への応用 — 神殿や劇場の隅角を正確に作図

  4. 証明の例 — 幾何学が「実験的観察」ではなく「論証的体系」であることを示す

現代のカリキュラムでこれを学ぶ意味#

1. 数学的思考の歴史#

  • 「定理」がどのように生まれたのか？

  • 経験的観察 → 仮説 → 証明 という過程

2. 計算思考の根源#

  • 円の作図 → 依存関係の構造化 → プログラミング

  • Geometric Scene = Variable Scope（幾何学的構造 = 変数のスコープ）

3. Dual Coding Theory#

  • ビジュアル（GeoGebra で見る）+ 記号（Python で計算）

  • 両方の表現を組み合わせることで、深い理解が生まれる

4. 実装可能性#

  • ggblab を使って、GeoGebra と Python を統合

  • 古代の定理を、モダンな計算環境で「再発見」する体験

次のステップ#

このチャプターでは以下の順序で学習を進めます：

  1. 01_intro.ipynb（このノートブック） → Thales の背景、定理の概要

  2. 01_draw_circle.ipynb → 黒板でコンパスを使って円を描く（視覚的直感）

  3. 01_geogebra.ipynb → GeoGebra で動的に作図（操作的理解）

  4. 01_protocol.ipynb → Construction Protocol で層構造を明示（構造化）

  5. 01_verify.ipynb → ggblab で自動検証（計算的検証）

  6. 01_theorem.ipynb → 定理を複数の方法で証明（概念統合）

それぞれが前のステップを深め、新しい視点を加えます。

思考問題#

以下の質問に答えてみてください：

  1. 「なぜ古代ギリシャの数学者は円に興味を持ったのか？」

    • 実用的な理由？美的な理由？哲学的な理由？

  2. 「Thales の定理は、どのような場面で有用か？」

    • 古代：建築、天文学

    • 現代：コンピュータグラフィックス、CAD？

  3. 「ビジュアル（GeoGebra）で見ることと、Python で計算することは、どう違うのか？」

    • それぞれの強み・弱みは？

    • 両方を組み合わせる価値は？
